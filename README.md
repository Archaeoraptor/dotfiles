# dotfiles

备份一份自用的dotfiles，包括zsh配置、bash配置、alacritty、yakuake、KDE主题定制、字体配置、vscode配置、vim配置（太菜了，vim用的不多，基本和默认设置差不太多）等

以及Nvidia独显驱动配置、螃蟹网卡、双显示器配置（DPI一个4k一个1080p）

（现在用的optimus-mamager切换显卡)
